# Team Tasks

**Team Tasks** is a dashboard designed for modern managers to organize teams, track real-time project progress, and leverage AI to stay ahead of the curve. It provides a distraction-free environment for high-level oversight.

<img src="./public/team-tasks-mockup.png"/>

---

## Key Features

- **Secured Workspace**: Role-based access and secure authentication powered by NextAuth, supporting both traditional credentials and Google OAuth.
- **Team Directory**: Effortlessly create, organize, and manage multiple teams with dedicated task boards.
- **Task Boards**: A sleek, bento-style task board for tracking deadlines and project stages (Not Started, In Progress, Completed).
- **AI Summaries**: Instant, automated team activity and project status reports generated by Google Gemini AI.

---

## Tech Stack

- **Framework**: [Next.js 15/16](https://nextjs.org/) (App Router)
- **Database**: [Prisma ORM](https://www.prisma.io/) with [MongoDB](https://www.mongodb.com/)
- **Authentication**: [NextAuth.js](https://next-auth.js.org/)
- **AI Engine**: [Google Generative AI (Gemini)](https://ai.google.dev/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/) (Phosphor, Material, Remix)

## Getting Started

### 1. Prerequisites
- Node.js 18.x or later
- A MongoDB cluster (Atlas or local)
- Google Gemini API Key

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/Manav437/Team-Tasks

# Install dependencies
npm install

# Generate Prisma client
npx prisma generate
```

### 3. Environment Setup
Create a `.env` file in the root directory and add the following variables:
```env
DATABASE_URL="your_mongodb_connection_string"
NEXTAUTH_SECRET="your_nextauth_secret"
GOOGLE_CLIENT_ID="your_google_id"
GOOGLE_CLIENT_SECRET="your_google_secret"
GEMINI_API_KEY="your_gemini_api_key"
```

### 4. Run Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to explore the project.
